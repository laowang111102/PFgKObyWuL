# 基于Flask的图书管理系统 python823

## 项目概述

本项目是一个基于 Flask 框架、使用 Python 语言并结合 MySQL 数据库的图书管理系统 Web 程序。该系统功能全面，提供了图书的增删改查等管理功能，能够满足大作业的需求，并获得了导师的高度评价。

## 技术栈

- 后端：Python 3.x
- Web 框架：Flask
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript（可选）

## 功能模块

- **用户认证**
  - 管理员登录功能

- **图书管理**
  - 图书的增加、删除、修改、查询
  - 图书分类管理

## 系统角色

- 管理员：负责登录系统进行图书管理操作

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x
- 操作系统：兼容 Windows、Linux、MacOS

## 部署步骤

1. 安装 Python 3.x
2. 安装 Flask：`pip install flask`
3. 安装数据库 MySQL 5.x
4. 创建数据库和相应的表
5. 运行 Flask 程序

## 目录结构

```
project/
│
├── app/                   # Flask 应用目录
│   ├── __init__.py        # Flask 应用初始化
│   ├── models.py          # 数据库模型
│   ├── views.py           # 视图处理
│   └── templates/         # HTML 模板目录
│
├── migrations/            # 数据库迁移脚本目录
│
├── venv/                  # 虚拟环境目录（推荐使用）
│
├── requirements.txt       # 依赖列表
│
└── run.py                 # 应用启动脚本
```

- 基于轻量级的 Flask 框架，易于维护和扩展。
- 实现了图书的完整管理流程，满足日常使用需求。
